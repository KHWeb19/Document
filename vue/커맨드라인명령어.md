# 정리를 습관화합시다!

```make
1. cd(Change Directory) 명령어
   위치 바꾸는데 사용함
2. dir 명령어
   현재 위치에 무엇이 있는지 보여줍니다.

* 조합
  - 현재 위치에 무엇이 있는지 살펴보고
  - 내가 이동하고 싶은 위치로 이동한다.
  ex) dir
  A   B   C   D   test   lec   gogogo
  
  cd test <<<--- dir로 뭐가 있는지 확인하고 이동하고 싶은 test 디렉토리로 이동함
  
* 상위 이동이 하고 싶은 경우
  - .. (상위 이동)
  ex) cd .. <<<--- 상위 디렉토리로 이동합니다.
```
