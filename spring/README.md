# Spring Boot 설정법

Spring 아니고 최신의 따끈따끈한 Spring Boot!  

## 기본 셋팅

```make
1. https://start.spring.io/ 접속
2. Project: Gradle Project
3. Language: Java
4. Spring Boot: 2.6.3
5. 나머지는 디폴트값으로 맞추도록 한다.
6. Add Dependencies
   6-1) Lombok
   6-2) Spring Web
   6-3) Thymeleaf
7. GENERATE를 누른다.
8. Spring에서 open하고 대기후 실행하면 8080 포트가 열리는 것을 볼 수 있다.
```
